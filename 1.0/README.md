# 用户信息管理平台（漏洞演示版）

> **⚠️ 警告：本项目仅供安全教学演示使用，包含大量高危漏洞，严禁用于生产环境！**

## 项目概述

一个基于 Python Flask 的简易用户信息管理平台，提供用户登录、信息展示和登出功能。本项目刻意保留了多个安全漏洞，用于安全教学和漏洞分析演练。

## 环境配置（Kali Linux）

```bash
cd /opt/Class01/1.0

# 检查环境
./setup.sh

# 启动服务
python3 app.py
```

如果 Flask 缺失，用 apt 安装：
```bash
sudo apt update && sudo apt install -y python3-flask
```

服务默认运行在 `http://0.0.0.0:5000`，所有网络接口均可访问。

## 项目结构


## 功能说明

- **首页 /** — 显示当前登录用户的信息，未登录则提示登录
- **登录 /login** — GET 显示登录表单，POST 验证凭据
- **登出 /logout** — 清除 session 并跳转首页
- 登录成功后，用户的**完整信息（含密码明文）** 会展示在页面上
